.idea - настройки, есть не у всех
out - скомпилированные файлы
.gitignore - создать и перечислить, что не будет попадать в проект. Всегда лежит в корне проекта

gitignore.io - сайт, чтоб создать гитигнор (генерирует данные)

git add .gitignore - добавить файл
git add . - добавить все

git rm --cached .gitignore - удалить отслеживаение 
git reset -  - удалить отслеживаение 


shift + F6 - быстрор переименовать класс джавы в интеледжи 

git commit -m "пояснение сюда писать" - фиксация изменений

git config -  настройки
git log
git branch -a
git push -u origin master - апстрим, нужен в первый раз, когда не было комитоов раньше



…or create a new repository on the command line
echo "# first_lesson" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/DoraSigulia/first_lesson.git
git push -u origin main

…or push an existing repository from the command line
git remote add origin https://github.com/DoraSigulia/first_lesson.git
git branch -M main
git push -u origin main
